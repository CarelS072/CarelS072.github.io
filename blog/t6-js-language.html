I was asked to "buy butter. See if they have eggs. If they do, buy 10" I returned with 10 packs of butter because they had eggs.

In this post I will go deeper into some of the interpunction that is used in JavaScript (JS) and what they get used for.

parentheses () Function - Whatever you put into the parentheses can be used as a placeholder to pass along into that piece of automation. Example: I can tell you to go to the supermarket and get a bottle of milk and some eggs. Or I can write these down on a piece of paper and give it to you to bring to the supermarket.
Loop - within the parentheses you will describe what the conditions for that loop will be in order for it to work. Example: When making instant noodles. You already have put the dried noodles in a bowl and you will need to boil the water first before you add water to it. If you do not do this, the instant noodles will not be instant.
brackets [] This is to highlight a collection of data (array). Think of it like a notebook in which you can write something down on page 1, 2, 3, etc. That pagenumber can then be used as a reference to the information it contains
braces {} Within the braces you will add piece of automation (code) to which will be "run" when it is called upon
single quotes '' Single quotes can be used if you want to describe something in a literal sense if it is something not natural in the code. When typing the code, often you will use something which already exists within the programming language. However when this doesn't exist quotes (or double quotes) can be used. Often this will be used for names or spaces if there are multiple words involved but they are a part of 1 being. When using Single quotes, it will have an opening and a closing single quote and cannot be interchanged with double quotes. IE you cannot close a single quote with a double quote.
double quotes "" Double quotes can be used if you want to describe something in a literal sense if it is something not natural in the code. When typing the code, often you will use something which already exists within the programming language. However when this doesn't exist quotes (or double quotes) can be used. Often this will be used for names or spaces if there are multiple words involved but they are a part of 1 being. When using double quotes, it will have an opening and a closing double quote and cannot be interchanged with single quotes. IE you cannot close a single quote with a double quote.
